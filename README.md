# SQL Server 資料庫開發實務

這是我整理的 SQL Server 基礎與進階練習專案，內容涵蓋資料庫設計、資料表建立、各種查詢語法與進階資料分析技巧。

透過這些範例，我練習了從資料建模到複雜查詢的實務開發流程。

## 📌 範例內容包含：

1. 資料表建立與資料插入
2. 基礎查詢語法(選擇、過濾、排序)
3. 聚合函數與群組查詢  
4. 多表關聯查詢(JOIN各種型態)
5. 子查詢與巢狀查詢
6. 視圖(VIEW)建立與更新  
7. CTE(Common Table Expression)用法  
8. Window Function應用  
9. 遞迴CTE與移動平均等進階查詢技巧
